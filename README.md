# phine-apps.github.io

Introduction page for phine-apps.

## Overview

This project is a cinematic landing page expressing the brand identity of `phine-apps`.
Built with Vite, React, Framer Motion, and Tailwind CSS v4.

## Features

- **Cinematic Animations**: Φ (Phi) symbol animation and StarField background powered by `framer-motion`.
- **Modern Styling**: Sleek dark-themed design using Tailwind CSS v4.
- **High Performance**: Fast build and development experience with Vite.
- **Auto Deployment**: Automated CI/CD pipeline for GitHub Pages via GitHub Actions.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 7
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Deployment**: GitHub Pages (Actions)

## Setup

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
